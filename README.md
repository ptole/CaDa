# Castle Darkholm
Silly simple roguelike with cheesy 80's B-movie flavour.
[Play it here!](https://ptole.github.io/CaDa/)

Made with 100% organic and additive-free [vanilla.js.](http://vanilla-js.com/)

#### Assets
Music and SFX by yours truly and sprites are literally just [unicode emojis.](https://unicode.org/emoji/charts/emoji-list.html)
